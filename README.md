# 📡 Satellite Signal Performance Analysis Dashboard

![Satellite Banner](images/satellite-banner.jpg)

## 📖 Overview
This project focuses on analyzing satellite communication performance using key parameters such as **Predicted Signal Strength (dBm)**, **Elevation Angle**, and environmental factors.

The objective is to derive meaningful insights that can improve signal reliability and communication efficiency in satellite systems.

- 📊 Signal Strength Analysis (dBm)
- 📡 Elevation Angle Impact Study
- 📈 Data Visualization using Tableau
- 🔍 Performance Trend Analysis
- ⚡ Engineering Insights for Communication Optimization

## ⚡ Engineering Insight
Satellite communication performance is significantly influenced by the **elevation angle** and atmospheric conditions.

- Lower elevation angles → Higher signal attenuation  
- Higher elevation angles → Better signal strength  

This project demonstrates how **data-driven analysis** can enhance communication efficiency and reliability.
